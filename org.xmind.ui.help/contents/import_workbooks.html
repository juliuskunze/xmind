Collaborative Minds

XMind file is a workbook containing a few sheets. Sometime, we may need to merge a few files as one. For example, organize team's report. Now, XMind supports it. The steps are very simple:

  1. Launch XMind and open the A.xmind file.
  2. Choose "Tools -- Import XMind Workbook"
  3. Select the B.xmind file and confirm.
  4. Then all sheets in B will be part of A.
