The summary is a brief description of topic areas on the map and is used to highlight the major points and help the audience quickly grasp the purpose of the map topics.

You can easily add, edit, modify, and delete a summary in XMind. Here's how:

To add a Summary:

  1. Select the targeted topics to add to the summary.
  2. You can use following methods to create a summary:
    • Choose "Insert > Summary" from the menu.
    • Use shortcut key "Ctrl+]" to add a summary.
    • Right-click with your mouse, and choose "Insert > Summary" from the context menu.
    • Enter the selected content in summary topic.
    • You can create the subtopics within the summary topic by clicking the "Tab" key.
    • Move the slides at the top and bottom of summary box to change the scope of the summary.

To change summary properties:

  1. Select a summary.
  2. Open the Properties view.
  3. You can change the following properties:
    • Summary Line Style, and color.
    • Summary Topic text formatting, Line, box shape, Structure and etc.

To delete an existing summary:

  1. Select a summary.
  2. Click "Delete" on the keyboard.
