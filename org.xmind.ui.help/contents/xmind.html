XMind

Collaborative Minds

XMind has four editions: XMind, XMind Plus, XMind Pro and XMind Pro Subscription. XMind is an open source software, which won "The Best Project for Academia" at Sourceforge.net Community Choice Awards 2009. XMind Plus/Pro/Subscription, based on XMind 3, are the commecial software packages. They incorporates some advanced features, such as Brainstorming mode, Presentation mode, Audio Notes, Task Info, Gantt View, Powerful Filter, Search, and etc.

They have the same installation package and are both free to be downloaded on our official website. While launching XMind, you are required to sign in with an XMind account. You can free sign up one. With this free account, we can:

  1. Share the mind maps with the World
  2. Upload mind maps without number limitation
  3. Watch and Comment others' maps
  4. Keep some beautiful mind maps via "Adding Favority" online or Downloading them
  5. Or, you can even has a private online space to store mind maps or share them with others secretly.

Of course, you can use XMind without signin. Then, you can not use the features of XMind Pro 3.

XMind can work fluently on Windows, Mac, and Linux. Cross platform with full features can liberate us perfectly. But this is not enough, XMind offers the real portable version, which

  1. is a zip file.
  2. can begin to work after being unzipped to a clear folder.
  3. can run even be placed in a U-disk.
  4. is a cross platform version too.

No matter XMind, or XMind Pro, it's easy to Draw your first map:

  1. Launch XMind/XMind Pro
  2. XMind will create a workbook with the central topic
  3. Click "F2" to type your own central topic
  4. Click "Enter" to create "Main Topic"
  5. Normally, "Enter" key is used to create topics at the same level. "Tab" key is used to create subtopics.
  6. To format the map, please open the "Properties View". Select a topic, and change its properties.
  7. No matter what feature you want to use, you can find the usage tips here.
